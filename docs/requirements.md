## 1. プロジェクト概要

### 1.1 プロジェクト名
**社内ヘルプデスク AI エージェント**
（変更が必要な場合はここに別の名前を記入）

### 1.2 背景・課題

👉 【私個人への属人化されたヘルプデスク対応。AIに聞けば一般的な回答は来るが、社内独自の仕様は私の脳にしかないものが多い。
　　　私が不在時にトラブルが発生した場合、わかる人がいないのが問題。ただし！システム管理者の特権情報を回答してはいけない】

```
現状の課題：
- ヘルプデスク業務が __社員名　システム管理者 に集中している
- 具体的な問題：MSソフトの使い方、PC操作、基幹システム操作対応、PC故障対応等

影響：
- その社員が不在の場合：業務停止
- その他の悪影響：業務停止
```

### 1.3 プロジェクトの目的

👉 【属人化された社内独自のヘルプデスク対応継承するが、システム特権に関わるものは変わらず非公開とする】

```
このプロジェクトで達成したいこと：
1. 属人化されたヘルプデスク対応をAIエージェントに継承
2. システム管理者特権は変わらず非公開とすること
3. 
```
